A Modern re-imagining / spiritual successor to qDirStat/kDirStat/WinDirStat/WizTree written in Rust, and highly paralellized.

While NTFS MFT scanning (Which WizTree uses to great effect on windows to scan at blazing fast speeds) isnt really translatable to linux filesystems there were still a multitude of ways the Linux applications could be optimized.
This is heavily Vibe-coded. You have been warned. So basically, expect nothing, enjoy what you get.

open in file browser, delete, refresh. MFT scanning support for Linux (only for NTFS drives) and a Windows build with the same.
